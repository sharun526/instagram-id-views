Updated Website Demo
Features:
- Profile pictures (SVGs included)
- Animated cards with hover effects
- Admin panel to add/remove Instagram IDs
- Live update simulation of status lines
- Smooth UI animations and transitions

How to use:
1. Unzip the package.
2. Open index.html in any modern browser.
3. Click 'Admin Panel' to add or remove Instagram IDs.
4. The list updates live and the cards animate on entry and hover.

Note:
- This is a static demo. Data is persisted locally using localStorage.
- Profile pictures are included as SVG files in the images/ folder.
